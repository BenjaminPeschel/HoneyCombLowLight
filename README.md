# HoneyCombLowLight
A full pipeline for removing the honey comb artifacts from nasal endoscopy and low light noise.
You can read up on the method in the following Journal of Voice (J Voice) paper:
"Reducing Fiber-Induced Honeycomb Artifacts and Low-Light Noise in Nasal High-Speed Video Laryngoscopy: A Fast, Deterministic, Open-Source Approach"
# Notes
write_to_file, writes into uncompressed, raw .avi files, which are huge. This was done deliberately to make the code easy to replicate. Nevertheless, I would recommend to change the codec, to allow compression, based on your individual needs
